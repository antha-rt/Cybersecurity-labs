# SheSecures México 2025 – Capture The Flag

This section contains sanitized write-ups from challenges completed during the **SheSecures México 2025 CTF** hosted on the Hackrocks platform.

The event focused on hands-on cybersecurity exercises across multiple technical domains and was supported by international and regional organizations promoting cybersecurity education.

## Challenge Categories

Challenges included topics such as:

- Cryptography
- Web application security
- Cloud security
- OSINT
- Digital forensics

Each challenge required analyzing provided datasets, identifying vulnerabilities or hidden information, and applying technical or investigative techniques to obtain the solution.

## Write-ups

The write-ups in this folder describe the analytical process used to solve each challenge while intentionally omitting sensitive details such as flags or exploit paths.

The goal is to document:

- Investigative methodology
- Vulnerability identification
- Reasoning process used during the challenge

## Skills Practiced

Through these challenges, the following skills were exercised:

- Web application analysis
- Metadata inspection
- Identifying encoded data
- Parameter manipulation
- Credential discovery
- OSINT and geolocation analysis

These exercises simulate investigative workflows commonly used in security analysis and incident response.
