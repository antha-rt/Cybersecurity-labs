# Cryptography / Data Encoding Challenge

## Objective:

A forensic dataset contained an image file recovered from a removable drive belonging to a research observatory. Hidden within the file was a message believed to reveal a key piece of information related to the investigation.

----

## Approach

Performed metadata inspection on the provided image file.

Identified suspicious encoded text embedded in the metadata fields.

Extracted the encoded string and analyzed the encoding scheme.

Determined the data was Base64-encoded and decoded it.

----

## Result

Decoding the hidden metadata revealed the embedded key required to complete the challenge.

---

## Skills demonstrated

- File metadata analysis

- Identifying encoded data patterns

- Base64 decoding

- Basic digital forensics workflow
