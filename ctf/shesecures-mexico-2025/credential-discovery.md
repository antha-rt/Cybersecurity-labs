# Cloud / Credential Discovery

## Objective

Investigate an authentication system suspected to contain a backdoor and identify a method to gain access.

---

## Approach

Reviewed client-side source code and application behavior.

Identified references suggesting previous successful logins.

Located exposed credential data in the application interface.

Used the discovered credentials to authenticate into the system.

---

## Result

Successful authentication granted access to protected content containing the challenge key.

---

## Skills demonstrated

- Web application analysis

- Credential discovery

- Understanding of authentication vulnerabilities

- Security implications of exposed credentials
