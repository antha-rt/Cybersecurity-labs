# Cybersecurity Labs

This repository documents hands-on cybersecurity training, labs, and challenge write-ups completed as part of my learning and professional development in security analysis

The goal of this repository is to document investigation methods, analytical reasoning, and technical techniques used during security exercises and challenges.

## Contents

### Capture The Flag (CTF)

Write-ups from cybersecurity challenge events involving multiple security domains such as:

- Web security
- OSINT
- Cryptography
- Cloud security
- Digital forensics

CTF challenges focus on identifying vulnerabilities, analyzing systems, and extracting hidden information through investigative techniques.

### TryHackMe Training

Notes and summaries from completed TryHackMe rooms related to:

- SOC analyst workflows
- Cyber threat intelligence
- Attack frameworks
- Security monitoring technologies (SIEM, EDR)
- Alert triage and incident reporting

The focus is on understanding defensive security operations and the analytical processes used in security monitoring and incident investigation.


## Purpose

This repository serves as a personal record of hands-on cybersecurity learning and practical exercises, with emphasis on:

- Investigative thinking
- Understanding attacker behavior
- Identifying vulnerabilities
- Applying structured security frameworks
